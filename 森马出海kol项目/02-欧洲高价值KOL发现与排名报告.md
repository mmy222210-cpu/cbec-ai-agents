# 森马欧洲高价值 KOL 发现与契合度排名报告

## 一、项目概述

**目标**：为森马 2026 春夏男装欧洲市场推广，发现高价值 KOL 并进行契合度排名
**预算**：$50,000
**参考技能**：influencer-discovery（发现框架）、fit-scorer（评分模型）、competitor-tracker（竞争情报）

## 二、发现参数

### 2.1 搜索条件

| 参数 | 要求 |
|------|------|
| 地区 | 欧洲（英国、德国、法国、意大利、西班牙、北欧） |
| 语言 | 英语、德语、法语（内容语言） |
| 品类 | 男装时尚、穿搭、街头风格、生活方式 |
| 粉丝范围 | 50K - 2M |
| 互动率 | >3%（高于行业均值） |
| 受众年龄 | 18-35 岁男性为主 |
| 内容调性 | 平价时尚、性价比穿搭、日常风格 |

### 2.2 排除条件

- 奢侈品为主的 KOL（品牌调性不符）
- 女性受众占比 >60% 的账号
- 近期有快时尚竞品独家合约的 KOL
- 争议性内容创作者

## 三、契合度评分模型

### 3.1 评分维度与权重

基于 fit-scorer 技能框架，针对森马品牌目标定制权重：

| 维度 | 权重 | 评分标准（1-5 分） |
|------|------|-------------------|
| **受众匹配** | 30% | 受众年龄、性别、地区与目标市场的重叠度 |
| **品牌契合** | 25% | 内容调性、价值观、价格定位与森马的一致性 |
| **内容质量** | 20% | 制作水平、创意能力、内容一致性 |
| **互动质量** | 15% | 互动率、评论质量、粉丝活跃度 |
| **合作潜力** | 10% | 过往品牌合作经验、沟通配合度、长期合作意愿 |

### 3.2 评分等级

| 评分范围 | 等级 | 建议 |
|---------|------|------|
| 4.5-5.0 | 卓越 | 必须联系，优先签约 |
| 4.0-4.49 | 优秀 | 高度推荐，积极联系 |
| 3.5-3.99 | 良好 | 推荐联系，可作为备选 |
| 3.0-3.49 | 合格 | 可考虑，视预算情况 |
| <3.0 | 不推荐 | 不建议合作 |

### 7.2 差异化策略

**森马的 KOL 合作差异化定位**：

```
SHEIN：     量大、低价、快消 ←──── 森马要避免
H&M/Zara：  大牌感、季节大片 ←──── 森马无法直接竞争
Uniqlo：    功能性、极简      ←──── 可部分借鉴

森马定位：   真实、有温度、性价比之王
             ↓
策略：选择以"真实推荐"著称的 KOL
     用对比内容证明性价比
     强调"不需要大价钱也能穿出风格"
```
